Semantic Search (의미론적 검색)

정의: 사용자의 질의를 단순한 키워드 매칭이 아니라 문맥과 의미를 분석하여 관련 정보를 반환하는 검색 방식.
예시: "우주 탐사"를 검색하면 "아폴로 11호", "화성 탐사 로버"와 같은 연관 정보가 포함된 결과를 제공함.
연관 키워드: 자연어 처리, 검색 알고리즘, 데이터 마이닝

FAISS (Facebook AI Similarity Search)

정의: FAISS는 페이스북에서 개발한 고속 유사성 검색 라이브러리로, 특히 대규모 벡터 집합에서 유사 벡터를 효과적으로 검색할 수 있도록 설계되었습니다.
예시: 수백만 개의 이미지 벡터 중에서 비슷한 이미지를 빠르게 찾는 데 FAISS가 사용될 수 있습니다.
연관키워드: 벡터 검색, 머신러닝, 데이터베이스 최적화

Embedding (임베딩)

정의: 단어나 문장을 벡터 공간에 매핑하여 의미적으로 유사한 것들이 가까이 위치하도록 하는 기법.
예시: "강아지"와 "고양이"의 벡터 표현이 유사하게 위치함.
연관 키워드: 벡터화, 자연어 처리, 딥러닝

Token (토큰)

정의: 텍스트 데이터를 더 작은 단위(단어, 문자, 문장 등)로 나누는 과정.
예시: "AI는 혁신적이다"를 ["AI", "는", "혁신적", "이다"]로 분할.
연관 키워드: 토큰화, NLP, 텍스트 전처리

Transformer (트랜스포머)

정의: 자연어 처리에서 사용되는 신경망 아키텍처로, 병렬 연산과 장기 의존성 처리가 강점.
예시: GPT, BERT 등의 모델이 트랜스포머 기반으로 동작함.
연관 키워드: 딥러닝, 자기 주의 메커니즘, NLP

Self-Attention (자기 주의 메커니즘)

정의: 문장의 모든 단어가 서로에게 가중치를 부여하여 문맥을 이해하는 방식.
예시: "나는 강아지를 좋아한다"에서 "나는"과 "좋아한다"가 강한 연관성을 가짐.
연관 키워드: 트랜스포머, BERT, 문맥 학습

Fine-Tuning (미세 조정)

정의: 사전 학습된 모델을 특정 작업에 맞게 추가 학습하는 과정.
예시: GPT 모델을 법률 문서 요약에 맞게 학습.
연관 키워드: 전이 학습, 모델 최적화, AI 응용

Zero-shot Learning (제로샷 학습)

정의: 특정 태스크에 대한 데이터 없이도 기존 학습된 지식을 활용하여 예측하는 AI 기술.
예시: AI가 새로운 언어를 학습하지 않았지만, 문맥을 기반으로 번역 가능.
연관 키워드: 전이 학습, 일반화 능력, NLP

Few-shot Learning (퓨샷 학습)

정의: 적은 양의 데이터로도 새로운 작업을 수행할 수 있도록 하는 학습 방법.
예시: ChatGPT가 몇 개의 예제만으로도 새로운 질문에 적절한 답을 생성함.
연관 키워드: 샘플 효율성, 전이 학습, 메타 학습

Reinforcement Learning (강화 학습)

정의: 보상을 극대화하는 방향으로 행동을 학습하는 AI 기법.
예시: 알파고가 바둑에서 최적의 수를 찾기 위해 강화 학습을 활용.
연관 키워드: 보상 시스템, 정책 학습, 게임 AI

Prompt Engineering (프롬프트 엔지니어링)

정의: AI 모델이 원하는 출력을 생성하도록 입력을 최적화하는 기술.
예시: "한 문장으로 요약해줘"라는 입력을 통해 짧은 답변을 유도.
연관 키워드: 자연어 생성, GPT, 입력 최적화

Large Language Model (대형 언어 모델, LLM)

정의: 대량의 텍스트 데이터를 학습한 자연어 처리 모델.
예시: GPT-4, PaLM 등이 대형 언어 모델에 해당.
연관 키워드: 자연어 처리, 트랜스포머, AI 모델

Vector Database (벡터 데이터베이스)

정의: 데이터를 벡터 형태로 저장하고 유사도를 기반으로 검색하는 데이터베이스.
예시: ChromaDB를 사용하여 유사 문서를 검색.
연관 키워드: 의미론적 검색, NLP, 임베딩

Latent Space (잠재 공간)

정의: 데이터가 저차원 벡터로 변환된 공간으로, 의미적으로 유사한 데이터들이 가까이 위치함.
예시: "강아지"와 "고양이"가 같은 클러스터에 배치됨.
연관 키워드: 차원 축소, 임베딩, 머신러닝

Generative AI (생성형 AI)

정의: 새로운 텍스트, 이미지, 음악 등을 생성하는 AI 기술.
예시: DALL·E가 주어진 텍스트 프롬프트를 기반으로 이미지를 생성.
연관 키워드: 생성 모델, GAN, 트랜스포머

Diffusion Model (확산 모델)

정의: 이미지 생성 AI에서 노이즈를 점진적으로 제거하며 데이터를 생성하는 기법.
예시: Stable Diffusion이 텍스트에서 이미지를 생성하는 과정.
연관 키워드: 이미지 생성, 딥러닝, 생성 AI

Hallucination (환각)

정의: AI가 실제 존재하지 않는 정보나 잘못된 내용을 생성하는 현상.
예시: AI가 가짜 논문을 만들어내는 경우.
연관 키워드: 신뢰성, 모델 평가, 자연어 처리

RAG (Retrieval-Augmented Generation, 검색 기반 생성)

정의: LLM이 외부 데이터베이스에서 정보를 검색하여 응답의 정확도를 높이는 기법.
예시: LangChain을 활용한 문서 검색 기반 AI 챗봇.
연관 키워드: 지식 검색, 하이브리드 AI, LangChain

Multimodal AI (멀티모달 AI)

정의: 텍스트, 이미지, 오디오 등 여러 유형의 데이터를 동시에 처리하는 AI 기술.
예시: GPT-4가 이미지와 텍스트를 동시에 이해하는 기능.
연관 키워드: 복합 데이터, AI 융합, 딥러닝

Explainable AI (설명 가능한 AI, XAI)

정의: AI의 의사결정 과정이 사람에게 이해할 수 있도록 설명되는 기술.
예시: AI가 의료 진단을 내릴 때, 왜 그런 결론을 내렸는지 설명함.
연관 키워드: 신뢰성, 모델 해석, AI 윤리

AGI (Artificial General Intelligence, 범용 인공지능)

정의: 특정 작업이 아니라 인간처럼 다양한 지적 작업을 수행할 수 있는 AI 개념.
예시: 현재 AI는 특정 작업에 특화된 ANI(특정 인공지능)이지만, 미래에는 AGI 개발이 목표.
연관 키워드: 자율 학습, 강한 AI, 미래 기술
